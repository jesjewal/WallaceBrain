# Workflow — InDesign Scripting

## Snapshot Rule
Always snapshot the target page **before AND after** making changes to visually verify results. Changes to geometry can have unintended side effects (overlapping items, clipped text).

## Direction vs. Coordinates
Infer all technical implementation from high-level design direction. Don't ask for pica coordinates or font names — derive them from existing page items. Jesse gives direction like "move the image left so the person is in frame" — translate intent into coordinates by inspecting existing items first.

## Brand/Font Consistency
New pages must match the established type system exactly. For Shive-Hattery: **Montserrat ExtraBold** for titles, **Bold** for labels, **Regular** for body.
